## 🔄 Миграция

### Обновление с 0.9.0 на 1.0.0

```typescript
// ДО (0.9.0)
const shield = new FABShield({
  headers: true,
  csp: true
})

// ПОСЛЕ (1.0.0)
const shield = new FABShield({
  headers: { enabled: true },
  csp: { enabled: true }
})
```

---

### Обновление с 0.5.0 на 1.0.0

```typescript
// ДО (0.5.0)
const shield = new Shield()

// ПОСЛЕ (1.0.0)
const shield = new FABShield({
  headers: { enabled: true }
})
```
