## 🎯 Категории угроз

### 1. Входные угрозы (Input Threats)

#### XSS (Cross-Site Scripting)

| Атрибут | Значение |
|:---|:---|
| **Описание** | Внедрение вредоносного JavaScript-кода |
| **Вектор атаки** | Параметры запроса, форма, URL |
| **Влияние** | Кража данных, сессий |
| **Защита** | CSP, Экранирование, AI-анализ |

**Пример:**
```html
<script>alert('XSS')</script>
SQL Injection
Атрибут	Значение
Описание	Внедрение SQL-кода в запросы
Вектор атаки	Параметры, тело запроса
Влияние	Доступ к данным, удаление данных
Защита	AI-анализ, Валидация
Пример:

sql
' OR '1'='1' --
CSRF (Cross-Site Request Forgery)
Атрибут	Значение
Описание	Выполнение действий от имени пользователя
Вектор атаки	Поддельный запрос
Влияние	Несанкционированные действия
Защита	CSRF-токены, SameSite cookies
DDoS (Distributed Denial of Service)
Атрибут	Значение
Описание	Перегрузка сервера запросами
Вектор атаки	Множество запросов
Влияние	Недоступность сервиса
Защита	Rate Limiting, AI-анализ
