## 📋 Введение

FAB Shield предоставляет **множество опций** для тонкой настройки безопасности. В этом руководстве мы подробно разберем все параметры конфигурации.

Правильная настройка позволит:
- 🎯 **Оптимизировать** защиту под ваши нужды
- ⚡ **Улучшить** производительность
- 🔧 **Интегрироваться** с существующими системами

---

## 🏗️ Структура конфигурации

```typescript
interface ShieldConfig {
    // Основные настройки
    enabled?: boolean
    env?: 'development' | 'production' | 'test'
    
    // Security-заголовки
    headers?: HeaderConfig
    
    // CSP
    csp?: CSPConfig
    
    // AI-защита
    ai?: AIConfig
    
    // Мониторинг
    monitoring?: MonitoringConfig
    
    // Плагины
    plugins?: Plugin[]
    
    // Кастомные правила
    rules?: CustomRule[]
    
    // Логирование
    logging?: LoggingConfig
    
    // Производительность
    performance?: PerformanceConfig
}
🔒 Настройка Security-заголовков
Базовая конфигурация
typescript
const shield = new FABShield({
    headers: {
        // Включить все заголовки (по умолчанию)
        enabled: true,
        
        // Отключить конкретные заголовки
        disabled: ['X-Powered-By', 'X-XSS-Protection'],
        
        // Кастомные заголовки
        custom: {
            'X-Custom-Header': 'custom-value',
            'X-Security-Level': 'high'
        }
    }
})

Настройка Referrer-Policy
typescript
const shield = new FABShield({
    headers: {
        referrerPolicy: {
            enabled: true,
            policy: 'strict-origin-when-cross-origin'
        }
    }
})
Доступные политики Referrer-Policy:
Политика	Описание
no-referrer	Не отправлять реферер
no-referrer-when-downgrade	Отправлять только при HTTPS → HTTPS
same-origin	Отправлять только на том же домене
origin	Отправлять только схему и домен
strict-origin	Отправлять только схему и домен при HTTPS → HTTPS
origin-when-cross-origin	Полный реферер на том же домене, только домен на других
strict-origin-when-cross-origin	Более строгая версия
unsafe-url	Отправлять полный URL (небезопасно)
🛡️ Настройка CSP (Content Security Policy)
Базовая конфигурация
typescript
const shield = new FABShield({
    csp: {
        // Включить CSP
        enabled: true,
        
        // Режим отчетности (только логи, без блокировки)
        reportOnly: false,
        
        // Динамический CSP (рекомендуется)
        dynamic: true,
        
        // Доверенные CDN
        trustedCDNs: [
            'https://cdnjs.cloudflare.com',
            'https://cdn.jsdelivr.net',
            'https://unpkg.com'
        ],
        
        // Кастомные директивы
        directives: {
            'default-src': ["'self'"],
            'script-src': ["'self'", "'unsafe-inline'"],
            'style-src': ["'self'", "'unsafe-inline'"],
            'img-src': ["'self'", "data:", "https:"],
            'font-src': ["'self'", "https:", "data:"],
            'connect-src': ["'self'", "https://api.example.com"],
            'frame-ancestors': ["'none'"],
            'base-uri': ["'self'"],
            'form-action': ["'self'"]
        }
    }
})
Расширенная конфигурация CSP
typescript
const shield = new FABShield({
    csp: {
        enabled: true,
        dynamic: true,
        reportOnly: false,
        
        // Отчеты о нарушениях
        reportUri: '/api/csp-report',
        reportTo: 'csp-endpoint',
        
        // Динамические директивы
        dynamicDirectives: {
            'script-src': (req) => {
                const nonce = generateNonce()
                return [`'nonce-${nonce}'`]
            },
            'style-src': (req) => {
                return ['https:', "'unsafe-inline'"]
            }
        },
        
        // Доверенные источники
        trustedCDNs: [
            'https://cdnjs.cloudflare.com',
            'https://cdn.jsdelivr.net'
        ],
        
        // Разрешить инлайн-скрипты с nonce
        nonceEnabled: true,
        
        // Разрешить инлайн-стили
        unsafeInlineStyles: false,
        
        // Строгий режим
        strict: true
    }
})

🔌 Настройка плагинов
typescript
const shield = new FABShield({
    plugins: [
        // WAF интеграция
        {
            name: 'waf-integration',
            enabled: true,
            config: {
                provider: 'cloudflare',
                apiKey: 'xxx'
            },
            middleware: async (req, res, next) => {
                // Логика WAF
                next()
            }
        },
        
        // Geo-блокировка
        {
            name: 'geo-blocking',
            enabled: true,
            config: {
                blockedCountries: ['RU', 'BY', 'CN']
            },
            middleware: async (req, res, next) => {
                const country = await getCountry(req.ip)
                if (blockedCountries.includes(country)) {
                    res.status(403).send('Access denied')
                    return
                }
                next()
            }
        },
        
        // Аудит
        {
            name: 'audit-logger',
            enabled: true,
            config: {
                logPath: './logs/audit.log',
                includeHeaders: false,
                includeBody: false
            },
            middleware: (req, res, next) => {
                console.log(`${new Date().toISOString()} ${req.method} ${req.url}`)
                next()
            }
        }
    ]
})
📝 Настройка кастомных правил
typescript
const shield = new FABShield({
    rules: [
        // Блокировка по IP
        {
            path: '*',
            action: 'block',
            condition: (req) => {
                const blockedIPs = ['192.168.1.100', '10.0.0.50']
                return blockedIPs.includes(req.ip)
            }
        },
        
        // Ограничение по методу
        {
            path: '/api/*',
            method: 'DELETE',
            action: 'block',
            condition: (req) => !req.headers['x-admin-token']
        },
        
        // Защита админки
        {
            path: '/admin/*',
            action: 'challenge',
            condition: (req) => {
                return !req.session?.isAdmin
            }
        },
        
        // Rate limiting для API
        {
            path: '/api/*',
            action: 'rate-limit',
            condition: (req) => true,
            config: {
                maxRequests: 100,
                windowMs: 60000
            }
        }
    ]
})

const shield = new FABShield({

                maxFiles: 5
            },
            {
                type: 'remote',
                enabled: false,
                url: 'https://logs.company.com'
            }
        ],
        
        // Что логировать
        include: {
            requests: true,
            threats: true,
            errors: true,
            performance: true,
            metrics: true
        },
        
        // Исключить чувствительные данные
