## 📊 Примеры потоков

### Пример 1: Обычный запрос
GET /api/users

Request → FAB Shield

Headers → Добавлены security-заголовки

AI Analysis → Безопасно (score: 0.1)

Threat Detection → Не найдено угроз

Action → ALLOW

Application → Обработка

Response → 200 OK

Metrics → Записаны

text

### Пример 2: Атака
POST /api/login (SQL инъекция)

Request → FAB Shield

Headers → Добавлены security-заголовки

AI Analysis → Подозрительно (score: 0.9)

Threat Detection → SQL Injection

Action → BLOCK

Response → 403 Forbidden

Metrics → Угроза записана

Alert → Уведомление отправлено
