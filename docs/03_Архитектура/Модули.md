## 📦 Описание модулей

### 1. 🧠 Core Module (Ядро)

**Назначение:** Базовый модуль, управляющий всем фреймворком.

**Ответственность:**
- Инициализация фреймворка
- Управление конфигурацией
- Координация других модулей
- Обработка жизненного цикла

**API:**
```typescript
class Core {
    constructor(config: ShieldConfig)
    init(): Promise<void>
    destroy(): Promise<void>
    getConfig(): ShieldConfig
    updateConfig(config: Partial<ShieldConfig>): void
}
2. 🔒 Headers Module
Назначение: Управление security-заголовками.

Ответственность:

Установка 25+ security-заголовков

Настройка HSTS

Настройка Referrer-Policy

Кастомные заголовки

API:

typescript
class HeadersModule {
    apply(req: Request, res: Response): void
    setHeader(name: string, value: string): void
    removeHeader(name: string): void
    getHeaders(): Record<string, string>
}
Поддерживаемые заголовки:

Content-Security-Policy

Strict-Transport-Security

X-Frame-Options

X-Content-Type-Options

Referrer-Policy

X-XSS-Protection

X-DNS-Prefetch-Control

X-Download-Options

X-Permitted-Cross-Domain-Policies

X-Powered-By (удаление)

И еще 15+ заголовков

🔄 Взаимодействие модулей
Схема взаимодействия
text
Запрос → Core → Headers → CSP → AI → Plugins → Rate → Metrics
                    ↓       ↓      ↓         ↓       ↓
                  Ответ   Ответ  Ответ    Ответ   Ответ
Поток данных
text
1. Core получает запрос
2. Headers устанавливает заголовки
3. CSP генерирует политику
4. AI анализирует запрос
5. Plugins выполняются
6. Rate Limiting проверяет лимиты
7. Metrics собирает данные
8. Ответ возвращается
📦 Независимые модули
Возможность использования по отдельности
Каждый модуль можно использовать независимо:

typescript
// Только заголовки
import { HeadersModule } from '@fab-registry/shield/headers'

const headers = new HeadersModule()
app.use(headers.middleware())

// Только AI
import { AIModule } from '@fab-registry/shield/ai'

const ai = new AIModule()
app.use(ai.middleware())

// Только CSP
import { CSPModule } from '@fab-registry/shield/csp'

const csp = new CSPModule()
app.use(csp.middleware())
