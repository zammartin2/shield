## 📋 Введение

**Официальные плагины** — это готовые расширения для FAB Shield, разработанные командой ООО «Деворбит». Они добавляют новую функциональность и интеграции, расширяя возможности фреймворка.

Все официальные плагины:
- ✅ **Полностью совместимы** с FAB Shield
- ✅ **Поддерживаются** командой разработчиков
- ✅ **Покрыты тестами**
- ✅ **Имеют документацию**
- ✅ **Бесплатны** и Open Source

## 🏗️ Список плагинов

### 1. 🔌 WAF Integration

**Назначение:** Интеграция с популярными WAF-сервисами.

**Поддерживаемые сервисы:**
- Cloudflare WAF
- AWS WAF
- Imperva WAF
- Akamai WAF
- ModSecurity

**Установка:**
```bash
npm install @fab-registry/plugin-waf
Использование:

typescript
import { wafPlugin } from '@fab-registry/plugin-waf'

const shield = new FABShield({
    plugins: [wafPlugin({
        provider: 'cloudflare',
        config: {
            apiKey: process.env.CLOUDFLARE_API_KEY,
            zoneId: process.env.CLOUDFLARE_ZONE_ID
        }
    })]
})
2. 🌍 Geo Blocking
Назначение: Блокировка трафика по географическому признаку.

Возможности:

Блокировка целых стран

Блокировка регионов

Разрешенные страны

Динамическое обновление списков

Установка:

bash
npm install @fab-registry/plugin-geo
Использование:

typescript
import { geoPlugin } from '@fab-registry/plugin-geo'

const shield = new FABShield({
    plugins: [geoPlugin({
        blockedCountries: ['RU', 'BY', 'CN', 'KP'],
        mode: 'block',  // 'block' | 'challenge' | 'monitor'
        allowedCountries: ['US', 'GB', 'DE', 'FR'],
        exceptions: {
            admin: ['RU'],  // Администраторам из RU разрешено
            trusted: ['*']  // Доверенным пользователям все разрешено
        }
    })]
})

7. 🛡️ IP Whitelist/Blacklist
Назначение: Управление списками IP-адресов.

Возможности:

Белый список

Черный список

Временная блокировка

Динамические списки

Установка:

bash
npm install @fab-registry/plugin-ip-list
Использование:

typescript
import { ipListPlugin } from '@fab-registry/plugin-ip-list'

const shield = new FABShield({
    plugins: [ipListPlugin({
        whitelist: ['10.0.0.1', '10.0.0.2', '192.168.1.100'],
        blacklist: ['1.2.3.4', '5.6.7.8'],
        mode: 'whitelist',  // 'whitelist' | 'blacklist'
        temporaryBlock: {
            enabled: true,
            duration: 3600,  // секунд
            attempts: 5
        },
        autoUpdate: {
            enabled: true,
            interval: 3600,
            url: 'https://api.company.com/ip-lists'
        }
    })]
})

            challenge: true,
            log: true

})

📦 Установка плагинов
Установка через npm
bash
# Установка конкретного плагина
npm install @fab-registry/plugin-waf

# Установка всех плагинов
npm install @fab-registry/plugins
Установка через реестр FAB
bash
npm install @fab-registry/plugin-waf --registry=https://fab.devorbit.ru
🚀 Использование нескольких плагинов
typescript
import { FABShield } from '@fab-registry/shield'
import { wafPlugin } from '@fab-registry/plugin-waf'
import { geoPlugin } from '@fab-registry/plugin-geo'
import { slackPlugin } from '@fab-registry/plugin-slack'
import { auditPlugin } from '@fab-registry/plugin-audit'

const shield = new FABShield({
    plugins: [
        wafPlugin({
            provider: 'cloudflare',
            config: {
                apiKey: process.env.CLOUDFLARE_API_KEY,
                zoneId: process.env.CLOUDFLARE_ZONE_ID
            }
        }),
        geoPlugin({
            blockedCountries: ['RU', 'BY', 'CN'],
            mode: 'block'
        }),
        slackPlugin({
            webhookUrl: process.env.SLACK_WEBHOOK_URL,
            channel: '#security'
        }),
        auditPlugin({
            logPath: './logs/audit.log'
        })
    ]
})

app.use(shield.middleware())
