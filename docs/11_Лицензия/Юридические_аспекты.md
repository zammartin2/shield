## 🔒 Конфиденциальность

### Сбор данных

FAB Shield **не собирает**:
- ❌ Персональные данные
- ❌ Данные пользователей
- ❌ Информацию о проектах

---

### Логирование

FAB Shield может вести логи для:

- 🔍 Отладки
- 📊 Мониторинга
- 🛡️ Безопасности

**Логи могут содержать:**
- IP-адреса
- Пути запросов
- Время запросов
- Статусы ответов

**Настройка логирования:**
```typescript
const shield = new FABShield({
    logging: {
        level: 'info',
        exclude: {
            headers: ['Authorization', 'Cookie'],
            body: ['password', 'token']
        }
    }
})
