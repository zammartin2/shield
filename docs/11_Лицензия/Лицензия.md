## 📋 Введение

**FAB Shield** распространяется по модели **Open Source Core + Commercial Cloud**.

- 🛡️ **Ядро (Core)** — распространяется под лицензией **MIT**. Полностью бесплатно.
- ☁️ **Облачные сервисы** — **Scout Cloud**, **Predictive AI**, **Enterprise** — коммерческие и требуют подписки.

Это стандартная и прозрачная модель, которую используют:

- **GitLab** — CE open-source, EE коммерческий
- **Redis** — open-source + Redis Cloud
- **Grafana** — OSS + Grafana Cloud
- **Sentry** — open-source + SaaS

## ☁️ Облачные сервисы

Облачные сервисы являются платной частью экосистемы FAB Shield.

---

### Scout Cloud

| Функция | Описание |
|:---|:---|
| **Хранение паттернов** | Безопасное облачное хранилище для паттернов атак |
| **Глобальный Threat Intelligence** | Доступ к обезличенным данным от всех клиентов |
| **Обучение AI** | AI-модель обучается на глобальных данных |
| **Автоматические обновления** | Shield получает новые правила из облака |

**Почему платно:**

- 💾 **Ресурсы** — серверы, базы данных, сеть
- ⚡ **Мощности** — GPU для обучения AI
- 🔒 **Безопасность** — шифрование, аудит, соответствие стандартам
- 📊 **Поддержка** — SLA 24/7

---

### Predictive AI / AI 4.0

| Функция | Описание |
|:---|:---|
| **Предсказание атак** | AI предсказывает атаки до их выполнения |
| **Поведенческий анализ** | Глубокий анализ поведения пользователей |
| **Нейросетевая защита** | Использование глубоких нейросетей |

**Почему платно:**

- 🧠 **Обучение моделей** — требует больших вычислительных мощностей GPU
- 📊 **Данные** — миллионы паттернов для обучения
- 🔄 **Постоянное обновление** — модели обновляются ежедневно
