## 📝 Подготовка к релизу

### Чек-лист перед релизом

#### 🔍 Проверка кода

- [ ] Все тесты проходят
- [ ] Покрытие тестами > 80%
- [ ] Линтер не выдает ошибок
- [ ] TypeScript компилируется
- [ ] Нет открытых критических багов
- [ ] Документация обновлена
- [ ] Примеры обновлены

#### 📦 Сборка

- [ ] Собрана production-версия
- [ ] Проверены все зависимости
- [ ] Проверена совместимость
- [ ] Проверен размер бандла

#### 🔒 Безопасность

- [ ] Проверка зависимостей (npm audit)
- [ ] Сканирование уязвимостей (Snyk)
- [ ] Проверка секретов
- [ ] Безопасность конфигурации

#### 📋 Документация

- [ ] Обновлен CHANGELOG.md
- [ ] Обновлен README.md
- [ ] Обновлена API документация
- [ ] Обновлены примеры
- [ ] Обновлены руководства

## 🚀 Процесс релиза

### 1. Обновление версии

```bash
# Обновление версии в package.json
npm version patch    # 1.0.0 → 1.0.1
npm version minor    # 1.0.0 → 1.1.0
npm version major    # 1.0.0 → 2.0.0

# Создание тега
git tag -a v1.0.0 -m "Release v1.0.0"
2. Обновление CHANGELOG.md
markdown
## [1.0.0] - 2026-07-01

### 🎉 Добавлено
- Первый стабильный релиз
- 25+ security-заголовков
- AI-защита
- Система плагинов
- Метрики и мониторинг

### 🛡️ Безопасность
- Исправлены все известные уязвимости
- Добавлена защита от XSS и SQL инъекций

### 📚 Документация
- Полная документация на русском
- Примеры использования
- API Reference
3. Сборка и тестирование
bash
# Очистка
npm run clean

# Установка зависимостей
npm ci

# Линтинг
npm run lint

# Проверка типов
npm run type-check

# Тесты
npm run test:ci

# Сборка
npm run build

# Проверка сборки
npm run preview
4. Публикация
bash
# Публикация в npm
npm publish

# Публикация с тегом beta
npm publish --tag beta

# Публикация в FAB Registry
npm publish --registry=https://fab.devorbit.ru

# Проверка публикации
npm view @fab-registry/shield
