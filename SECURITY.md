## 🚨 Сообщение об уязвимости

### Как сообщить

**Важно:** НЕ создавайте публичные Issue для уязвимостей!

1. **Напишите нам на email:**
derector@devorbit.ru

2. **В теме письма укажите:**
[SECURITY] FAB Shield Vulnerability

3. **В письме укажите:**
- Подробное описание уязвимости
- Шаги для воспроизведения
- Версию FAB Shield
- Ваши контактные данные (опционально)

### Что происходит после сообщения

✅ **Подтверждение (в течение 24 часов)**
→ Мы подтверждаем получение

🔍 **Расследование (1-5 дней)**
→ Мы анализируем уязвимость

🔧 **Исправление (до 30 дней)**
→ Мы разрабатываем патч

📢 **Раскрытие (после патча)**
→ Мы публикуем информацию

🛡️ Наши стандарты безопасности
1. Защита данных
✅ Все данные шифруются при передаче (TLS 1.2+)
✅ Пароли хешируются (bcrypt, salt rounds 10+)
✅ Токены подписываются (JWT, RS256)
✅ Секреты не хранятся в коде

2. Защита от атак
Тип атаки	Защита
XSS	CSP, защита заголовков, экранирование
CSRF	SameSite cookies, CSRF токены
SQL Injection	Параметризованные запросы
DDOS	Rate limiting, защита Cloudflare
MITM	TLS, HSTS, сертификаты
Brute Force	Rate limiting, блокировка IP
ReDoS	Валидация входных данных
3. Безопасность кода
