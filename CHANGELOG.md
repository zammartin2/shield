## Формат версий

Проект следует Semantic Versioning (SemVer):

- **MAJOR (2.0.0)** — несовместимые изменения API
- **MINOR (1.3.0, 1.4.0)** — добавление функциональности (обратно совместимо)
- **PATCH (1.0.x)** — исправления багов (обратно совместимо)
