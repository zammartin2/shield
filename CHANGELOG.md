## [1.3.5] - 2026-07-06

### 🔒 Security

- **Исправлена уязвимость в ConfigManager** (Socket.dev GPT Security — medium)
  - Проблема: загрузка конфигурационных файлов с расширениями `.js`, `.cjs`, `.mjs` через `require()` могла выполнять произвольный JavaScript-код при контроле `configPath` злоумышленником
  - Решение: 
    - Добавлена валидация пути конфигурации перед загрузкой
    - Улучшена проверка содержимого файлов конфигурации
    - Добавлена защита от prototype pollution
    - Ограничены разрешённые расширения для конфигурационных файлов
  - Дополнительно: усилена проверка целостности конфигурации через checksum
  - Статус: ✅ Уязвимость полностью устранена

### 🚀 Добавлено

- **Достигнуто 91.06% покрытия кода тестами** ✅ (+1.09% к предыдущему релизу)

  - `ResponseHandler.ts`

### 🧪 Тестирование
